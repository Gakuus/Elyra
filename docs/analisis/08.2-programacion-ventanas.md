# 8.2 - ProgramaciГіn вҖ” Bosquejo de Estructuras de Ventanas

## Estructura General del Sistema Elyra

El sistema se compone de **dos mГіdulos principales**, cada uno con su propio flujo de ventanas. A continuaciГіn se presentan los bosquejos de las estructuras de ventanas para cada mГіdulo.

## Convenciones de la Interfaz

| Elemento | Comportamiento |
|----------|---------------|
| Barra superior | Logo Elyra, nombre del mГіdulo, usuario autenticado, botГіn cerrar sesiГіn |
| BotГіn [вҶҗ Volver] | Presente en todas las pantallas internas |
| Tablas | Con paginaciГіn, ordenamiento por columnas, y bГәsqueda |
| Formularios | Con validaciГіn en cliente y servidor |
| Modales | Para confirmaciones de eliminaciГіn y cancelaciГіn |
| Estados de color | Verde = completado, Amarillo = pendiente, Rojo = cancelado, Azul = en curso |
| Responsive | Adaptable a desktop, tablet y mobile (Bootstrap 5) |
